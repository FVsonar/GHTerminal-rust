# GH-Terminal Docs

## Overview

GH-Terminal is a desktop control terminal (Tauri + Svelte) for GuoHe radio equipment, communicating over a serial port.

## Quick links

- Protocol spec: `doc/国赫电台设备控制协议.md`
- Workspace:
  - `gh-protocol/` (protocol codec + commands)
  - `gh-terminal/` (Tauri backend)
  - `frontend/` (Svelte UI)
